# SIGNAL

Android app for prioritizing and managing important notifications.

## Setup

1. Open the project in Android Studio or VS Code.
2. Create a local `local.properties` file in the project root.
3. Set your Android SDK path and API key values there.
4. Build with `./gradlew assembleDebug` or install to a connected device with `./gradlew installDebug`.

## Local properties

This repository keeps secrets out of git. Copy [local.properties.example](local.properties.example) to `local.properties` and fill in your values.
